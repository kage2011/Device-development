## 2026-03-10 RS485 1C通信の安定化メモ
- 成功の鍵は、ユーザー提供のPlatformIOコードの通信手順へ寄せることだった。
  - ENQ + "00FF" + body + CR
  - 制御文字(STX/ETX/ACK/NAK/CR)を除外して本文を復元
  - 14バイト枠(STX + "00FF" + 8hex + ETX)を優先採用
- Dレジスタ固定値検証で読取り成功を確認:
  - D0=100 -> 00FF0064
  - D10=200 -> ...00C8
  - D20=300 -> 00FF012C
  - D30=30000 -> 00FF7530
  - D40/D41で65536相当の応答確認
- D2000/D2001の復元は `u32 = (w2 << 16) | w1` (lo-hi) で整合。
- 20秒・0.1秒サンプリングを実施し、CSV/グラフ作成済み:
  - results/d2000_20s_0p1s.csv
  - results/d2000_20s_0p1s.png
